# 09 消費税・割引計算

ID=09 のアプリ「消費税・割引計算」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=09）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=9 のライト/ダーク色）

## 機能

- 定価入力
- 税率切替（8% / 10%）
- 割引率スライダー（0%〜90%）
- 割引後価格、税額、総額、値引き額の表示
- ライト・ダーク切替（テーマ保存）

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: 計算ロジック、イベント処理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `tax-discount.theme.v1`: テーマ設定
