# 02 チェックリスト

ID=02 のアプリ「チェックリスト」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=02: 項目追加、完了チェック、全削除、リセット）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=2 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- 項目追加（最大80文字）
- 完了チェックの切替
- フィルター（すべて / 未完了 / 完了）
- リセット（完了状態を一括解除）
- 全削除（確認ダイアログあり）
- ライト・ダーク切替（テーマ保存）
- ローカルストレージ永続化

## ファイル構成

- `index.html`: 画面構造（追加フォーム、一覧、操作）
- `styles.css`: チェックリスト専用スタイル
- `app.js`: 状態管理、永続化、描画、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `simple-checklist.items.v1`: 項目データ
- `simple-checklist.theme.v1`: テーマ設定
