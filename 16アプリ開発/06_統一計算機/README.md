# 06 統一計算機

ID=06 のアプリ「統一計算機」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=06: 四則演算、％計算、直近1件の計算履歴表示）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=6 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- 四則演算（＋, −, ×, ÷）
- ％計算
- 直近1件の履歴表示
- AC / DEL 操作
- ライト・ダーク切替（テーマ保存）

## ファイル構成

- `index.html`: 画面構造（表示部、キーパッド）
- `styles.css`: 計算機専用スタイル
- `app.js`: 計算ロジック、状態管理、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `unified-calculator.theme.v1`: テーマ設定
