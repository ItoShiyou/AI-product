# 08 割り勘計算機

ID=08 のアプリ「割り勘計算機」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=08: 総額・人数入力、端数丸め設定、結果表示）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=8 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- 総額入力
- 人数入力
- 端数丸め設定（なし / 切り上げ / 切り捨て / 四捨五入）
- 1人あたり金額表示
- 合計徴収額と差額表示
- ライト・ダーク切替（テーマ保存）

## ファイル構成

- `index.html`: 画面構造（入力、結果）
- `styles.css`: 割り勘計算機専用スタイル
- `app.js`: 計算ロジック、状態管理、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `split-bill.theme.v1`: テーマ設定
