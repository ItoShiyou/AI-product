step1
HTML5の Canvas API を使用して、診断結果の数値を反映した『Xシェア用画像（1200x630px）』を動的に生成するJavaScript関数 generateShareImage() を作成してください。

画像の要件:

背景: 現在のサイトデザインに合わせた、ダークでサイバーなグラデーション背景。

テキスト:

中央に巨大な赤字で『AI代替率：[replacementRate]%』。

その下に『職種余命：[lifespan]年』。

下部にサイト名『AI余命宣告』とURL。

装飾: テキストの周りに、近未来的なグリッチエフェクトやスキャンラインのテクスチャを重ねてください。

出力: 生成したCanvasを DataURL（Base64） として取得できるようにしてください。


################################################################################
step2
Xの Web Intent（https://twitter.com/intent/tweet） を使用して、診断結果をシェアするボタンを実装してください。

仕様:

text パラメーターには、先ほど作成した動的な投稿文（『私のAI代替率はXX%でした…』）を設定。

url パラメーターには、ユーザー固有の診断結果ページ（または、OGPタグが動的に書き換わる中間ページ）のURLを設定してください。

重要: XのクローラーがそのURLを読みに来た際、ステップ1で生成した画像が og:image として認識されるような、サーバーサイド（Vercel Functions等）のメタタグ書き換えロジックも併せて提案してください。


################################################################################
step3
診断結果画面に、ステップ2で実装したXシェアボタンを配置してください。

演出の要件:

プレビュー表示: ボタンの横に、ステップ1で生成された『実際に投稿される画像』の縮小版をプレビューとして表示し、ユーザーの投稿意欲を掻き立ててください。

マイクロコピー: ボタンの上に『【残酷な事実】をフォロワーに共有して、生存確率を10%上げる』という、ジョーク交じりのコピーを追加してください。

投稿後処理: シェアボタンがクリックされた後、数秒後にA8のアフィリエイトバナーを強調表示させる処理を追加してください。
