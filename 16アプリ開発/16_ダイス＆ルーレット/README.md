# 16 ダイス＆ルーレット

ID=16 のアプリ「ダイス＆ルーレット」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=16）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=16 のライト/ダーク色）

## 機能

- ダイスモード（面数指定）
- ルーレットモード（改行区切り項目編集）
- 乱数抽選
- 抽選中の簡易アニメーション表示
- 履歴表示（直近20件）
- 履歴クリア
- ルーレット項目の保存
- ライト・ダーク切替（テーマ保存）

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: モード切替、抽選ロジック、履歴管理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `dice-roulette.theme.v1`: テーマ設定
- `dice-roulette.items.v1`: ルーレット項目
