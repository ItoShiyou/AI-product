# 大富豪 Firebase 連携設計

## 目的
VS Code から Firebase を使った開発をすぐ始められるように、初期化コードと実行手順を整備する。

## 構成
- Firebase 初期化: `src/firebase.js`
- 画面エントリ: `src/main.js`
- 確認用ページ: `index.html`
- VS Code タスク: `.vscode/tasks.json`

## Firebase 設定
Firebase プロジェクトID: `daifugoh-fd702`

`src/firebase.js` に Firebase App 初期化を実装。
Analytics はブラウザかつサポート環境のみで有効化する。

## 開発手順
1. ターミナルで `npm install`
2. VS Code で `Tasks: Run Task` を開く
3. 初回のみ `Firebase: Login` を実行
4. エミュレータ利用時は `Firebase: Start Emulators`
5. ローカル表示確認は `npm run start`

## デプロイ手順
1. `Firebase: Deploy` タスクを実行
2. または `npm run firebase:deploy`
