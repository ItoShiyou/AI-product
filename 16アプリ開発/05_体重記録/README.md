# 05 体重記録

ID=05 のアプリ「体重記録」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=05: 数値入力、日付記録、簡易的な推移グラフ表示）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=5 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- 体重入力（20.0kg〜300.0kg）
- 日付記録（同日入力時は上書き更新）
- 推移グラフ表示（Canvas）
- 最新値と前回差分の表示
- 記録一覧表示
- 記録削除
- ライト・ダーク切替（テーマ保存）
- ローカルストレージ永続化

## ファイル構成

- `index.html`: 画面構造（入力、推移、一覧）
- `styles.css`: 体重記録専用スタイル
- `app.js`: 状態管理、永続化、描画、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `weight-tracker.records.v1`: 体重記録データ
- `weight-tracker.theme.v1`: テーマ設定
