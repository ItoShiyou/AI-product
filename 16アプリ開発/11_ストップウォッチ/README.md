# 11 ストップウォッチ

ID=11 のアプリ「ストップウォッチ」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=11）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=11 のライト/ダーク色）

## 機能

- 計測開始・停止
- ラップタイム記録
- リセット
- ラップ一覧表示
- ライト・ダーク切替（テーマ保存）

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: 計測ロジック、ラップ管理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `stopwatch.theme.v1`: テーマ設定
