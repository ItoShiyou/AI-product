# 07 単位変換器

ID=07 のアプリ「単位変換器」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=07: 単位選択（長/重/面等）、数値入力、相互変換）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=7 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- カテゴリ切替（長さ / 重さ / 面積）
- 変換元・変換先単位選択
- 数値入力による即時変換
- 単位入れ替え
- 変換式表示
- ライト・ダーク切替（テーマ保存）

## ファイル構成

- `index.html`: 画面構造（設定、結果）
- `styles.css`: 単位変換器専用スタイル
- `app.js`: 変換ロジック、状態管理、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `unit-converter.theme.v1`: テーマ設定
