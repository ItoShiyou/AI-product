# 13 QRスキャナー

ID=13 のアプリ「QRスキャナー」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=13）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=13 のライト/ダーク色）

## 機能

- カメラ起動/停止
- QRコード検出（`BarcodeDetector` 対応ブラウザ）
- URL検出時のリンク表示
- スキャン履歴の保存と一覧表示
- 履歴全削除
- ライト・ダーク切替（テーマ保存）

## 注意

- ブラウザが `BarcodeDetector` API に未対応の場合、QR検出は利用できません。
- HTTPS またはローカル環境でのカメラ権限が必要です。

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: カメラ制御、QR検出、履歴管理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `qr-scanner.theme.v1`: テーマ設定
- `qr-scanner.history.v1`: スキャン履歴
