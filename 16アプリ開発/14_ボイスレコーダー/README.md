# 14 ボイスレコーダー

ID=14 のアプリ「ボイスレコーダー」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=14）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=14 のライト/ダーク色）

## 機能

- 録音開始・停止
- 録音時間表示
- 録音リスト表示
- 各録音の再生
- 各録音のダウンロード保存
- 各録音の削除
- ライト・ダーク切替（テーマ保存）

## 注意

- マイク権限が必要です。
- 録音データはページリロードで消えます（ブラウザメモリ管理）。

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: MediaRecorder 制御、録音管理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `voice-recorder.theme.v1`: テーマ設定
