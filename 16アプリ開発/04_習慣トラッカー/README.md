# 04 習慣トラッカー

ID=04 のアプリ「習慣トラッカー」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=04: 習慣登録、ワンタップ達成記録、月間ログ表示）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=4 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`

## 機能

- 習慣登録（最大40文字）
- 今日の達成ワンタップ切替
- 月間ログ表示（日別ドット）
- 月切替（前月/翌月）
- 習慣削除
- ライト・ダーク切替（テーマ保存）
- ローカルストレージ永続化

## ファイル構成

- `index.html`: 画面構造（追加フォーム、月間ログ、一覧）
- `styles.css`: 習慣トラッカー専用スタイル
- `app.js`: 状態管理、永続化、描画、イベント処理
- `FUNCTIONS.md`: 主要関数の責務一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `habit-tracker.habits.v1`: 習慣データ
- `habit-tracker.theme.v1`: テーマ設定
