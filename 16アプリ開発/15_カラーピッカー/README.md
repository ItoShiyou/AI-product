# 15 カラーピッカー

ID=15 のアプリ「カラーピッカー」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=15）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=15 のライト/ダーク色）

## 機能

- カメラ開始/停止
- 映像のタップ位置ピクセルの色取得
- HEX/RGB 表示
- HEX のクリップボードコピー
- 最近の色履歴表示と再適用
- 履歴クリア
- ライト・ダーク切替（テーマ保存）

## 注意

- カメラ権限とクリップボードAPIが必要です。

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: カメラ制御、色抽出、コピー処理
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `color-picker.theme.v1`: テーマ設定
- `color-picker.recent.v1`: 最近の色履歴
