# 10 キッチンタイマー

ID=10 のアプリ「キッチンタイマー」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=10）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=10 のライト/ダーク色）

## 機能

- 分・秒の時間設定
- プリセット（1/3/5/10分）
- カウントダウン開始・一時停止・リセット
- 終了時アラーム音
- 通知APIによる完了通知（許可時）
- ライト・ダーク切替（テーマ保存）

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: タイマー制御、通知、アラーム
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `kitchen-timer.theme.v1`: テーマ設定
