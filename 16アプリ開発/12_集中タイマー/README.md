# 12 集中タイマー

ID=12 のアプリ「集中タイマー」です。

- 参照: `アプリ開発/アプリリスト.md`（ID=12）
- 参照: `アプリ開発/共通デザイン手法.md`
- 参照: `アプリ開発/カラーリスト.md`（ID=12 のライト/ダーク色）

## 機能

- ポモドーロ式の `集中 25分 / 休憩 5分`
- フェーズ切替（手動）
- 開始・一時停止・次フェーズスキップ・リセット
- 集中フェーズ完了サイクルの記録
- 完了時アラーム音と通知（許可時）
- ライト・ダーク切替（テーマ保存）

## ファイル

- `index.html`: 画面構造
- `styles.css`: アプリ固有スタイル
- `app.js`: タイマー状態管理、サイクル記録
- `FUNCTIONS.md`: 主要関数の責務

## 永続化キー

- `focus-timer.theme.v1`: テーマ設定
- `focus-timer.stats.v1`: 完了サイクル数
