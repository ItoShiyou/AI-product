# 03 — 簡易家計簿

## 概要

毎日の収入・支出をシンプルに記録し、月ごとの収支を把握するツールアプリ。

## 機能

| 機能 | 説明 |
|------|------|
| 記録追加 | 種別（収入/支出）・金額・項目・日付を入力して追加 |
| 月間サマリー | 表示月の収入合計・支出合計・収支差分をリアルタイム表示 |
| 月ナビゲーション | ‹ › ボタンで前後の月に切り替え |
| 日別リスト | 明細を日付ごとにグループ化して降順表示 |
| 記録削除 | 各明細の × ボタンで個別削除 |
| 自動保存 | ローカルストレージに自動保存。ブラウザを閉じてもデータが残る |
| ダークモード | Dark / Light ボタンでテーマを即時切替 |

## カラー

| モード | カラー | 名称 |
|--------|--------|------|
| ライト | `#B2D8B2` | Sage |
| ダーク | `#2E402E` | Deep Sage |

## 使い方

1. 「支出」または「収入」を選択する
2. 金額・項目を入力し（Enter キーで次フィールドへ移動）、日付を確認する
3. 「追加する」をタップ
4. 上部サマリーカードで月間収支を確認する
5. ‹ › で月を切り替えて過去の記録を閲覧する
