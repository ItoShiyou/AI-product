# 01 シンプルメモ

ID=01 のアプリ「シンプルメモ」です。以下の仕様書を参照して実装しています。

- `アプリ開発/アプリリスト.md`（ID=01: テキスト入力、自動保存、一覧表示、簡易検索）
- `アプリ開発/共通デザイン手法.md`
- `アプリ開発/カラーリスト.md`（ID=1 のライト/ダーク色）
- `アプリ開発/アプリ共通概要.md`（CSS変数化、共通UIクラス方針）

## 機能

- メモタイトル入力（最大60文字）
- メモ本文入力（最大500文字）
- タグ設定（仕事/個人/学習/買い物）
- メモ色分け（Sand/Mint/Sky/Rose/Slate）
- 下書き自動保存（`localStorage`）
- メモ保存と一覧表示
- キーワード検索 + タグ絞り込み
- メモ編集（既存メモを読み込み、上書き更新）
- メモ固定（ピン留めしたメモを常に上位表示）
- 並び替え（更新新しい順 / 更新古い順 / タイトル昇順）
- 日時表示切替（更新日時優先 / 作成日時優先）
- ごみ箱（誤削除から復元、完全削除、空にする）
- JSON書き出し/読み込み（置換または追加取り込み）
- ライト・ダーク切替（テーマ保存）

## 便利操作

- `Ctrl + Enter`（Macは `Cmd + Enter`）で本文入力中に保存

## ファイル構成

- `index.html`: 画面構造（ヘッダー、入力、一覧、ごみ箱）
- `styles.css`: デザイントークンと共通UIクラス
- `app.js`: メモデータ管理、描画、イベント処理
- `FUNCTIONS.md`: 主要関数の役割一覧

## ローカル実行

`index.html` をブラウザで開くと動作します。

## 永続化キー

- `simple-memo.list.v3`: メモ一覧（`title`/`body`/`tag`/`color`/`timestamps`）
- `simple-memo.list.v2`: 旧形式メモ一覧（互換読込のみ）
- `simple-memo.list.v1`: 旧形式メモ一覧（互換読込のみ）
- `simple-memo.trash.v1`: ごみ箱データ
- `simple-memo.draft.v3`: 入力中の下書き（`title`/`body`/`tag`/`color`）
- `simple-memo.draft.v2`: 旧形式下書き（互換読込のみ）
- `simple-memo.draft.v1`: 旧形式下書き（互換読込のみ）
- `simple-memo.prefs.v1`: 表示設定（並び替え/日時表示）
- `simple-memo.theme.v1`: テーマ設定

## 保守のポイント

- 保存形式を変更する場合は、キーのバージョン（例: `.v3`）を上げる。
- レンダリングは `render()` 経由で統一し、状態更新直後に呼び出す。
- 新機能追加時は `FUNCTIONS.md` の該当関数説明も更新する。
