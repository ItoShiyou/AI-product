    ルール

    このゲームはテキサスホールデムをベースにしています。

    基本ルール

    • 各プレイヤーに2枚の手札が配られます。
    • 共通カードは Flop（3枚）→ Turn（1枚）→ River（1枚）の順で公開されます。
    • 手札2枚と共通カード5枚から最強の5枚役で勝敗を決めます。
    • アクションは Fold / Check / Call / Raise です。
    • 役の強さ（上ほど強い）: ストレートフラッシュ / フォーカード / フルハウス / フラッシュ / ストレート / スリーカード / ツーペア / ワンペア / ハイカード

    ひとりで対戦モードのルール

    • プレイヤー1人と Bot 複数人で対戦します。
    • 各ハンド終了時に結果画面が表示され、配当移動を確認できます。
    • 次のハンドは「新しいハンド」ボタンで開始します。

    みんなで対戦モードのルール

    • 2〜8人で同じルームに参加して対戦します。
    • 待機部屋で 20 秒以上そろわない場合、空き枠に AI Bot が参加します。
    • 満室時は待機表示となりますが、Bot 枠はハンド終了後に人間へ置き換え可能です。
    • このモードは現在準備中で、今後の実装対象です。

    みんなで対戦モード（待機部屋）

    roomId を共有して同じ部屋に参加します。20秒待機でBot補充されます。
